This directory contains datasets for crop and fertilizer recommendations.

```
crop_recommendation/
├── soil_data_sample.csv
└── README.md
```

## Dataset Information

The crop recommendation dataset contains soil and environmental parameters that are used to recommend suitable crops and fertilizers.

## Data Fields

- `nitrogen` - Soil nitrogen content (ppm)
- `phosphorus` - Soil phosphorus content (ppm)
- `potassium` - Soil potassium content (ppm)
- `ph` - Soil pH level
- `temperature` - Average temperature (°C)
- `humidity` - Average humidity (%)
- `rainfall` - Annual rainfall (mm)
- `crop` - Recommended crop
